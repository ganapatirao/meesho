# Meesho Clone

A full-stack e-commerce application built with .NET Web API and React frontend.

## Project Structure

- **BusinessPlatform.API** - .NET Web API backend
- **BusinessPlatform.Frontend** - React frontend application

## Prerequisites

- .NET 8.0 SDK
- Node.js 18+ and npm/yarn
- MongoDB

## Getting Started

### Backend Setup

1. Navigate to the API directory:
```bash
cd BusinessPlatform.API
```

2. Restore dependencies:
```bash
dotnet restore
```

3. Update connection string in `appsettings.json`

4. Run the API:
```bash
dotnet run
```

### Frontend Setup

1. Navigate to the frontend directory:
```bash
cd BusinessPlatform.Frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Features

- User authentication
- Product browsing
- Shopping cart
- Category management
- Company management
