# Advertising - High Level Specification

## Overview
Users can browse classified ads, post their own ads, and contact sellers for various items and services.

## Core Features

### 1. Browse Ads
- View all active ads
- Filter ads by category
- Filter ads by subcategory
- Search ads by keywords
- View ad details (title, description, price, images)
- View seller contact information
- Sort ads by price or date posted

### 2. Post Ads
- Create new ad listings
- Upload ad images
- Set ad price
- Select category and subcategory
- Provide ad description
- Add seller contact information
- Edit own ads
- Delete own ads
- Activate/deactivate own ads

### 3. Ad Categories
- Electronics
- Vehicles
- Real Estate
- Services
- Jobs
- Fashion
- Home & Garden
- Others

### 4. Seller Interaction
- View seller contact details
- Send message to seller
- Call seller (if phone provided)
- Report inappropriate ads

### 5. My Ads
- View all posted ads
- Edit ad details
- Delete ads
- View ad status (active/inactive)
- View ad views/interest

## Business Rules
- User must be logged in to post ads
- Users can only edit/delete their own ads
- Ads require category and subcategory
- At least one image required
- Seller contact information mandatory
- Inactive ads not visible to others
- Ads can be reported for removal

## User Experience
- Category and subcategory filters
- Image gallery for ads
- Search functionality
- Contact seller directly
- Manage own ads easily
