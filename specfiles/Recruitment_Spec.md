# Recruitment - High Level Specification

## Overview
Users can browse job postings, apply for jobs, and employers can manage job applications.

## Core Features

### 1. Browse Jobs
- View all active job postings
- Filter jobs by location
- Filter jobs by job type (full-time, part-time, contract, internship)
- Search jobs by keywords or company
- View job details (title, company, description, salary, location)
- View required skills and qualifications
- View company information
- Sort jobs by date posted or salary

### 2. Job Application
- Apply for jobs
- Upload resume
- Add cover letter
- Submit application
- View application status
- Withdraw application
- Track multiple applications

### 3. My Applications
- View all submitted applications
- View application status (pending, reviewed, accepted, rejected)
- View application details
- Update application if allowed

### 4. Job Categories
- Technology
- Healthcare
- Finance
- Education
- Marketing
- Sales
- Engineering
- Others

## Business Rules
- User must be logged in to apply
- One application per job per user
- Resume upload required
- Cover letter optional but recommended
- Applications can be withdrawn before review
- Employers can view applications
- Application status updated by employer

## User Experience
- Location-based filtering
- Job type filtering
- Search by skills or keywords
- Easy application process
- Track application status
